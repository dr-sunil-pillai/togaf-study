# TOGAF Study Assistant — PWA

AI-powered TOGAF 10 exam preparation, powered by the T-Systems LLM Hub.

## Deploy to GitHub Pages (5 minutes, free)

This gives you a real URL your team can bookmark and install as a mobile app.

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
- Click **New repository**
- Name it: `togaf-study`
- Set to **Public**
- Click **Create repository**

### Step 3 — Upload the files
- Click **uploading an existing file**
- Drag and drop ALL files from this folder:
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `icons/` folder (all 8 icons inside)
- Click **Commit changes**

### Step 4 — Enable GitHub Pages
- Go to **Settings** → **Pages**
- Under Source, select **main** branch → **/ (root)**
- Click **Save**
- Wait ~60 seconds, then your app is live at:
  `https://YOUR-GITHUB-USERNAME.github.io/togaf-study/`

---

## Install on Android (Home Screen App)

1. Open Chrome on Android
2. Go to your GitHub Pages URL
3. Tap the **⋮ menu** (top right)
4. Tap **"Add to Home screen"**
5. Tap **"Install"** or **"Add"**

The app now appears on your home screen with the TOGAF icon.
Opening it launches fullscreen — no browser bar, just like a native app.

---

## Install on iPhone / iPad (Safari)

1. Open Safari and go to your URL
2. Tap the **Share button** (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **"Add"**

## First-time setup

When you first open the app, the ⚙️ Settings panel appears.
Enter your **T-Systems LLM Hub API key** and choose a model.
